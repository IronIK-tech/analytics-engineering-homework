# Homework 1 — Docker Compose + PostgreSQL

Setting up a local PostgreSQL environment using Docker Compose.

## Tasks
1. Create `docker-compose.yml` with PostgreSQL service, environment variables, custom port and volume
2. Start the container with `docker compose up -d`
3. Connect to the database via DBeaver or DataGrip
4. Create a table, insert records, run SELECT
5. Test data persistence — stop and restart containers
6. Test data removal — stop with `docker compose down -v` and restart

## Stack
- Docker / Docker Compose
- PostgreSQL
